DAY-01: Python Fundamentals

1. Input and Output

Input: Use input() to get user input.
Output: Use print() to display output.

print("Hello world")
Hello world
print("bibek")
bibek
print("1st program in python")
1st program in python
print("india", "odisha", "ganjam")
india odisha ganjam
print("india", "odisha", "ganjam", sep="-")
india-odisha-ganjam
print("Hello", end=" ")\nprint("world")
Hello world

2. Data Types

print(4)
4
print(1e309)
inf
print(True)
True
print(False)
False
print("4+j")
4+j

3. List, Tuple, Set, Dict

print([9,0,7,7,4,7])
[9, 0, 7, 7, 4, 7]
print((9,0,7,7,4,7))
(9, 0, 7, 7, 4, 7)
print({9,0,7,7,4,7})
{0, 9, 4, 7}
print({"name":"bibek","age":"20","sex":"male"})
{'name': 'bibek', 'age': '20', 'sex': 'male'}

4. Variable Assignment

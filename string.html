DAY-03: Python Strings Fundamentals

1. Creating Strings

  • String can be created by enclosing characters inside single (' '), double (" "), or triple quotes (''' ''' or """ """).
  • Triple quotes are used to represent multiline strings.
# Single and double quotes
c = 'Hello'
print(c)
c = "Hello"
print(c)

# Multiline string using triple quotes
c = '''Hello
World'''
print(c)

s = str('hello')
print(s)
Hello Hello Hello World hello

2. Accessing Substrings (Indexing and Slicing)

  • Indexing starts at 0 (positive index, left to right).
  • Negative indexing counts from right to left, starting at -1.
  • Trying to access index outside the string bounds raises an error.
  • Slicing allows extracting substring ranges: string[start:end]
c = "hello"
print(c[0])          # Positive Indexing
print(c[-1])         # Negative Indexing

c = "Hello World"
print(c)
print(c[0:5])        # First 5 chars
print(c[2:])         # From index 2 to end
print(c[:4])         # From start to index 4
print(c[:])          # Full string
print(c[0:8:3])      # Slicing with step
print(c[0:6:-1])     # No output (wrong slicing)
h o Hello World Hello llo World Hell Hello World HlW

3. Editing and Deleting Strings

  • Strings are immutable in Python.
  • You cannot edit parts of a string directly;
  • Strings can only be reassigned.

6. Common String Functions

  • len() - length of string
  • max() - max character
  • min() - min character
  • sorted() - sorted list of characters
c = "kolkata"
print(len(c))
print(max(c))
print(min(c))
print(sorted(c, reverse=True))
7 t a ['t', 'o', 'l', 'k', 'k', 'a', 'a']

9. Programming Problems on Strings

Find length of string without len()

s = input('Enter the string:')
counter = 0
for i in s:
  counter += 1
print('length of string is:', counter)
Enter the string: bibek length of string is: 5

Extract username from email

s = input('Enter the email:')
pos = s.index('@')
print(s[0:pos])
Enter the email: bibek4panda@gmail.com bibek4panda

Count frequency of a character in string

s = input('Enter the string:')
term = input('What would you like to search for:')
counter = 0
for i in s:
  if i == term:
    counter += 1
print('frequency', counter)
Enter the string: bibek What would you like to search for: b frequency 2

Remove a character from string

s = input('Enter the string:')
term = input('What would you like to remove:')
result = ''
for i in s:
  if i != term:
    result = result + i
print(result)
Enter the string: vicky What would you like to remove: y vick

Check Palindrome

s = input('Enter the string:')
flag = True
for i in range(len(s)//2):
  if s[i] != s[len(s)-i-1]:
    flag = False
    print('The string is not a Palindrome')
    break
if flag:
  print('The string is Palindrome')

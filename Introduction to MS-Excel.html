Introduction to MS-Excel

Main Components of the Interface

  1. Tabs: Located at the top (Home, Insert, Page Layout, etc.), each tab groups related commands.
  2. Ribbon: Toolbar displaying all commands/features needed to work in Excel.
  3. Groups: Commands in each tab are organized into functional groups.
  4. Column heading/column: Vertical lines, labeled A, B, C, etc.
  5. Row heading/row: Horizontal lines, numbered 1, 2, 3, etc.
  6. Range: A selection of two or more cells (e.g., A1:B10).
  7. Cells: Individual workspace boxes identified by row & column (e.g., A1).
  8. Quick Access Toolbar: Top-left toolbar for one-click access to common commands.
  9. Sheet Tabs: Tabs at the bottom to navigate between worksheets.
  10. Status Bar: Shows info about worksheet state (e.g., sum, mode).
  11. Zoom In: Slider or "+" icon to enlarge content display.
  12. Zoom Out: Slider or "-" icon to reduce content display.
  13. Minimize: Hides Excel to taskbar.
  14. Maximize: Expands Excel window full screen.
  15. Exit: Closes Excel or just the workbook.
  16. Name Box: Shows address or name of the selected cell/range.
  17. Formula Bar: Shows and edits selected cell content/formula.
  18. Workbook: The full Excel file containing sheets.
  19. Cells: Basic unit for data (text, numbers, formulas).
  20. Quick Access Toolbar: Customizable, above/below ribbon for fast access to your favorite commands.
  21. Worksheet: Individual spreadsheet page in a workbook.

Components and Features of a Worksheet

  • Rows: Horizontal, numbered, up to 1,048,576 rows in recent Excel.
  • Columns: Vertical, labeled A to XFD (up to 16,384 columns).
  • Cells: At row & column intersection (e.g., A1), holds data or formulas.
  • Cell Reference: Unique by column letter and row number, used in formulas.
  • Formula Bar: Enter/edit/view formulas or content of selected cell.
  • Worksheets: Multiple per workbook, switch with sheet tabs.
  • Navigation: Scrollbars/keys; "Go To" for fast navigation (Ctrl+G).
  • Gridlines: Faint lines separate cells, can be shown/hidden or printed.
  • Sheet Tabs: Add/delete/rename sheets by right-clicking tab.

Quick Access Toolbar (QAT)

  • Location: Top-left by default; can move below Ribbon.
  • Customizable: Add/remove any frequently used command.
  • One-Click Access: Fast access to commands, no menu hunting.
  • Adding Commands: Use dropdown arrow for a list, or via File > Options.
  • Removing Commands: Dropdown arrow or via File > Options.
  • Keyboard Shortcuts: Each QAT command gets an Alt + Number shortcut.

How to Customize the QAT

  • Add Commands: Click dropdown on QAT, select "More Commands" & pick from the list.
  • Move the QAT: Right-click QAT > Show Below the Ribbon.
  • Reset: File > Options > Quick Access Toolbar > Reset.

Dialog Box vs Task Pane

Feature Dialog Box Task Pane
Appearance Separate window pops up Dockable panel on window side
Interactivity Modal (blocks main window) Non-modal (can keep working)
Duration Closes after completing task Remains open as needed
Common Use Detailed, one-time settings/actions Ongoing/contextual tools
Examples Format Cells, Page Setup, Sort PivotTable Field List, Clipboard

Entering & Editing Data

Entering Data in Excel:
  • Open Excel and select a cell (e.g., A1).
  • Type data; press Enter/Tab to move to the next cell.
  • Multiple Data Entry: Use fill handle (drag to fill numbers/dates/series).
Editing Data:
  • Edit in cell: Double-click or select and type.
  • Edit in Formula Bar: Select cell, edit contents up top.
  • Cut/Copy/Paste: Select cells, right-click for options or use keyboard shortcuts.
  • Delete Data: Select cell(s), press Delete or right-click > Clear contents.
Editing Columns and Rows:
  • Resize: Drag borders of column/row headers.
  • Insert/Remove Rows or Columns:
    • Insert: Right-click header, select "Insert".
    • Delete: Right-click header, select "Delete".

Saving the Workbook

  • Save: File tab > Save or Save As.
  • AutoSave: For Office 365/cloud, AutoSave can keep work updated automatically.

Font Type & Size

To change the font type and size in Excel, you can follow these steps:

Changing Font Type:

  1. Select the cell or range of cells whose font you want to modify.
  2. Go to the Home tab in the Excel ribbon.
  3. In the Font group, you'll see a drop-down menu labeled "Font." Click on this drop-down menu.
  4. From the drop-down menu, select the desired font type.
  5. Alternatively, you can directly type the font name in the font box and press Enter.

Changing Font Size:

  1. Select the cell or range of cells whose font size you want to modify.
  2. Go to the Home tab in the Excel ribbon.
  3. In the Font group, you'll see a drop-down menu labeled "Font Size." Click on this drop-down menu.
  4. From the drop-down menu, select the desired font size.
  5. Alternatively, you can directly type the font size in the font size box and press Enter.
Keyboard Shortcuts:
To change font type: Select the cell or range and use Ctrl + Shift + F to open the font dialog box.
To change font size: Select the cell or range and use Ctrl + Shift + P to open the font size dialog box.

Bold, Italic & Underline

To apply bold, italic, and underlined formatting in Excel:
  1. Bold: Select cells — press Ctrl + B or Ctrl + 2 on your keyboard, or click the "B" icon in the toolbar.
  2. Italic: Select cells — press Ctrl + I or Ctrl + 3, or click the "I" icon in the toolbar.
  3. Underline: Select cells — press Ctrl + U or Ctrl + 4, or click the "U" icon in the toolbar.

Increase / Decrease Font Size

  • Increase Font Size: Ctrl + Shift + >
  • Decrease Font Size: Ctrl + Shift + <
  1. Select the cell or range of cells whose font size you want to modify.
  2. Press Ctrl + Shift + > to increase or Ctrl + Shift + < to decrease the font size.
Alternatively, you can use the ribbon:
  • Increase Font Size:
    • Select cells
    • Go to the Home tab
    • In the Font group, click the increase font size button (upward "A" arrow)
  • Decrease Font Size:
    • Select cells
    • Go to the Home tab
    • In the Font group, click the decrease font size button (downward "A" arrow)

Borders

Borders in MS Excel are used to enhance the appearance of cells by adding lines around them. Borders can help organize data, separate different sections, highlight important information, or make a table easier to read and understand.

Types of Borders:

  • Bottom Border: Adds a line to the bottom of the selected cell(s).
  • Top Border: Adds a line to the top of the selected cell(s).
  • Left Border: Adds a line to the left side of the selected cell(s).
  • Right Border: Adds a line to the right side of the selected cell(s).
  • All Borders: Adds lines to all four sides of the selected cell(s), creating a box around the data.
  • Outside Borders: Adds a border around the outer edges of the selected cell(s) but not inside the range.
  • Thick Outside Border: Adds a thicker border around the outer edges of the selected cell(s).
  • No Border: Removes any existing borders from the selected cell(s).
  • Border Styles and Colors: Excel provides options to customize border styles and colors (solid, dashed, dotted, double; different thicknesses; variety of colors).
  • Using the Format Cells Dialog Box: Select cells — right-click and choose Format Cells or press Ctrl + 1. Go to the Border tab to customize style, color, and placement (top, bottom, left, right, inside, outside).
  • Drawing Borders: Use the Draw Border tool for custom borders on your worksheet.
  • Gridlines vs. Borders: Gridlines are faint and for viewing only (not printed); borders print and are visible in shared files.
  • Removing Borders: Select cells, go to the border drop-down in the Font group, choose "No Border".

Fill Color

Regular Way to Fill Color in Cells

  1. Select the cell or range of cells in which you want to fill the color.
  2. Click the Home tab in the Excel ribbon.
  3. In the Font group, click on the Fill Color icon (looks like a paint bucket, near the text color icon).
  4. Click on your chosen color in the palette. If not available, click More Colors for more options (including hex/RGB).
  5. Once you've chosen your desired color, click Ok to apply.

Shortcut

The keyboard shortcut Alt + H + H in Excel opens the Fill Color panel for selected cells:

  1. Select the cell or range of cells.
  2. Press Alt, then H, then H again.
  3. The Fill Color panel opens—use arrow keys to choose a color, Enter to apply.
Pros of this shortcut:
  • Faster than using the mouse (once practiced).
  • Allows you to not switch from keyboard to mouse.
Cons of this shortcut:
  • Doesn’t allow typing a color; navigation is by arrow keys.
  • Slower than expected if unfamiliar.
  • Hard to remember.
